# Store Project

A simple e-commerce website built with HTML, CSS, and JavaScript.

## Overview

This project is a front-end e-commerce website developed as a portfolio project. It includes product browsing, shopping cart management, user authentication interfaces, product reviews, and rating functionality.

## Features

### Home Page

* Product listing
* Search functionality
* Navigation between pages

### User Authentication

* Login page
* Registration page
* User account management

### Product Page

* Product details
* Product image gallery
* Color selection
* Size selection
* Product rating system
* Customer reviews and comments
* Average rating calculation for each product

### Shopping Cart

* Add products to cart
* Remove products from cart
* Update product quantity
* Edit cart items
* Price calculation
* Cart management

### Data Storage

* LocalStorage support
* SessionStorage support

## Technologies Used

* HTML5
* CSS3
* JavaScript
